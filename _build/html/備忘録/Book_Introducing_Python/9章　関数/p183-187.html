関数の引数で見かける「*args」,「**kwargs」,「*」¶

投稿日：2022年2月14日
最終更新日：2022年2月14日

参照 入門 Python 3 第2版 オライリー・ジャパン pp.183-187

ドキュメントを読んでいると、関数の引数に「*args」,「**kwargs」,「*」といった文字を見かけます。「*args」は可変個の実引数をとることができます。「**kwargs」は複数のキーワード引数を1個の辞書にまとめることができます。「*」はキーワード専用引数を指定できます。

「*args」 *による位置引数¶

def print_args(*args):
    print("Positional tuple:", args)

関数呼び出しの実引数を複数指定することができる。複数の引数は関数内でタプルとして扱われる。

print_args(1, 2, 3, "よん")

Positional tuple: (1, 2, 3, 'よん')

*aとして渡すとタプルのアンパックが行われる。

a = (1, 2, 3, 4)
print_args(a)
print_args(*a)

Positional tuple: ((1, 2, 3, 4),)
Positional tuple: (1, 2, 3, 4)

「**kwargs」 **によるキーワード引数¶

def print_kwargs(**kwargs):
    print("Keyword arguments:", kwargs)

複数のキーワード引数を渡すことができる。複数のキーワード引数は関数内で辞書として扱われる。

print_kwargs(wine="merlot", entree="mutton", dessert="macaroon")

Keyword arguments: {'wine': 'merlot', 'entree': 'mutton', 'dessert': 'macaroon'}

データ型が辞書であるbを関数に渡すときは**bとする。

b = {'wine': 'merlot', 'entree': 'mutton', 'dessert': 'macaroon'}
print_kwargs(**b)

Keyword arguments: {'wine': 'merlot', 'entree': 'mutton', 'dessert': 'macaroon'}

bをそのまま渡すとエラーを返される。

print_kwargs(b)

TypeError                                 Traceback (most recent call last)

----> 1 print_kwargs(b)

TypeError: print_kwargs() takes 0 positional arguments but 1 was given

「*」 キーワード専用引数¶

*以降のキーワード引数は関数呼び出し時にキーワードが必須になる。

def print_data(data, *, start=0, end=100):
    for value in (data[start:end]):
        print(value)

data = [1, 2, 3, 4, 5]
print_data(data)

1
2
3
4
5

print_data(data, start=1, end=3)

2
3

関数呼び出し時にキーワードを指定することが必須であるため、以下の例では実行できない。

print_data(data, 1, 3)

---------------------------------------------------------------------------
TypeError                                 Traceback (most recent call last)
<ipython-input-11-580e69399918> in <module>()
----> 1 print_data(data, 1, 3)

TypeError: print_data() takes 1 positional argument but 3 were given

以下の例はキーワード専用引数がないため、関数呼び出し時にキーワードを指定しなくとも実行できる。

def print_data(data, start=0, end=100):
    for value in (data[start:end]):
        print(value)

print_data(data, 1, 3)

2
3
