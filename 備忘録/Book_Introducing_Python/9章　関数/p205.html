再帰を利用したリストのリストのリストの平坦化¶

投稿日：2022年2月15日
最終更新日：2022年2月15日

参照 入門 Python 3 第2版 オライリー・ジャパン p.205

Python 3.3 で追加されたyield fromを使用する。これはジェネレータがほかのジェネレータに仕事の一部を委ねることができる。

def flatten(lol):
    for item in lol:
        if isinstance(item, list):
            yield from flatten(item)
        else:
            yield item

lol = [1, 2, [3,4,5], [6,[7,8,9],[]]]
flatten(lol)

<generator object flatten at 0x7fc1ac98bdd0>

list(flatten(lol))

[1, 2, 3, 4, 5, 6, 7, 8, 9]
